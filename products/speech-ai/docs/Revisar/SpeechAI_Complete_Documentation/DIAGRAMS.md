# Diagrams Index

This document serves as an index for all architectural and flow diagrams associated with the Speech AI project. The actual diagram files are stored in the `../Diagrams/` directory.

## 1. System Architecture

- **File**: `../Diagrams/system_architecture.puml` (PlantUML)
- **Description**: High-level component diagram showing the relationship between the App Kernel, Pipeline, Intelligence Engine, and Providers.

## 2. Pipeline Data Flow

- **File**: `../Diagrams/pipeline_flow.mmd` (Mermaid)
- **Description**: Sequence diagram detailing the step-by-step transformation of raw text into a final audio file.

## 3. Class Structure (Domain Models)

- **File**: `../Diagrams/domain_models.puml` (PlantUML)
- **Description**: UML Class diagram illustrating the `Presentation`, `Slide`, `Paragraph`, and `SpeechProfile` entities.

*(Note: Diagram source files are provided in the release package and can be rendered using standard PlantUML or Mermaid rendering tools).*
