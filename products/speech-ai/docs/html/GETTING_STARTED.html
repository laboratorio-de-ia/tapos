🚀 Como Começar com a Documentação Speech AI

Bem-vindo! Este guia ajudará você a navegar pela documentação e encontrar exatamente o que precisa.

📖 O que é Speech AI?

Speech AI é uma plataforma empresarial de Text-to-Audio que transforma scripts em áudio de alta qualidade. Desenvolvida com arquitetura modular, oferece suporte a múltiplos idiomas (Inglês, Português, Espanhol) e integração com diversos provedores TTS.

🎯 Características Principais

Característica Descrição
🌍 Multi-Idioma Detecção automática de idioma e seleção de voz apropriada
🧠 Inteligência Análise de complexidade de texto e otimização de narração
🔌 Extensível Suporte para múltiplos provedores TTS (Edge, Azure, OpenAI)
📦 Modular Componentes desacoplados com responsabilidades claras
🚀 Escalável Arquitetura pronta para produção em larga escala

❓ Perguntas Frequentes

P: Como faço para começar a desenvolver?

R: Acesse o Developer Pack e siga o Developer Guide. Ele tem instruções passo-a-passo para setup local.

P: Onde encontro informações sobre segurança?

R: Veja o Operations Pack → Security Guide para postura de segurança, ameaças e mitigações.

P: Como é a arquitetura do sistema?

R: Leia Foundation Pack → Architecture e veja os diagramas em Diagrams.

P: Qual é o roadmap do projeto?

R: Consulte Sprint Pack → Roadmap para a visão de produto com 3 fases e timelines.

P: Como faço para contribuir?

R: Veja Developer Pack → Contributing para workflow de contribuição, branching strategy e processo de PR.

P: Como faço deploy em produção?

R: Siga o Operations Pack → Operations Guide e Release Process.

P: Posso usar a documentação offline?

R: Sim! Todos os links são relativos. Baixe a pasta Docs_HTML e abra index.html no seu navegador.

P: Como atualizo a documentação?

R: Edite os arquivos Markdown em SpeechAI_Docs/ e execute convert_md_to_html.py para regenerar os HTMLs.
