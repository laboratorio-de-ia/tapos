Speech AI - Enterprise Text-to-Audio Platform

Overview

Speech AI is a modern, scalable, and modular Text-to-Audio (TTS) platform built with Python. It transforms written scripts in multiple languages (English, Portuguese, Spanish) into high-quality, natural-sounding audio files. Designed with an enterprise-grade architecture, it acts as a robust pipeline capable of analyzing text, extracting intelligence, optimizing speech patterns, and orchestrating synthesis through extensible providers.

This documentation suite serves as the single source of truth for the project, covering architecture, development standards, operations, and strategic roadmaps.

Core Capabilities

  • Multi-Language Support: Automatically detects and processes English, Portuguese, and Spanish.
  • Speech Intelligence Engine: Analyzes text complexity, calculates pacing, and recommends optimal reading styles and speeds.
  • Provider Abstraction: A factory and registry pattern allows seamless switching between TTS providers (currently using Microsoft Edge TTS, with planned support for Azure, OpenAI, ElevenLabs, and Amazon Polly).
  • Voice Profiles: Configuration-driven voice selection based on language, gender, and reading style (voices.json).
  • Modular Pipeline: Decoupled stages for Text Analysis, Narration Building, Speech Building, and TTS Generation.

Getting Started

To run the Speech AI pipeline locally:

  1. Install Dependencies: bash pip install -r requirements.txt

  2. Configure Input: Place your text file in the input/ directory (e.g., input/script.txt). Update config/settings.json if necessary.

  3. Run Pipeline: bash python main.py

  4. Retrieve Output: Check the output/ directory for the generated .mp3 file and text reports.
